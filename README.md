MeteoApp - Application Météo Python

MeteoApp est une application météo développée en Python qui permet aux utilisateurs d'obtenir des prévisions météorologiques précises pour leur emplacement actuel ou pour n'importe quelle ville dans le monde. L'application utilise des API de services météorologiques pour récupérer les données en temps réel et les afficher de manière conviviale.
Fonctionnalités principales :
- Affichage des prévisions météorologiques actuelles et à venir.
- Recherche de prévisions pour n'importe quelle ville.
- Interface utilisateur simple et intuitive.
- Utilisation d'API de services météorologiques pour des données précises.
